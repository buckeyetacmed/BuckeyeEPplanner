# EP Medical Pre-Plan — Buckeye TacMed

Executive Protection Medical Pre-Planning application by Buckeye TacMed.

## Deployment

1. Upload all files to this repository root
2. Enable GitHub Pages (Settings → Pages → Deploy from branch → main)
3. Access at `https://[username].github.io/[repo-name]/`

## Files

- `index.html` — Main application
- `ep-manifest.json` — PWA manifest (home screen install)
- `ep-sw.js` — Service worker (offline support)
- `ep-icon-192.png` — Home screen icon (192×192)
- `ep-icon-512.png` — Home screen icon (512×512)

## Adding to Home Screen (iOS)

1. Open the app URL in Safari
2. Tap the Share button
3. Tap "Add to Home Screen"
4. Tap "Add"
